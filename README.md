# ubunomo

(Ub)untu based Docker image with (No)de.js and (Mo)ngoDB inside.

## Info

* Based on [bitbar/ubuno](https://github.com/bitbar/ubuno) which has:

  * Ubuntu LTS (18.04)
  * Node.js LTS (10)
  * [learn more](https://github.com/bitbar/ubuno)

* MongoDB 4.2
